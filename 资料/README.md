# 资料

解答面试题的硬核资料 = **优质开源 Agent 的真实源码** + **源码导读**。

- [`agent-frameworks.md`](agent-frameworks.md) — 15 个优质 Agent 源码索引（含语言、一句话定位、面试看点）。源码本体在 `life-os/intake/github-trending/agent-frameworks/`，本目录通过 `agent-frameworks` 软链接接入。
- [`源码导读.md`](源码导读.md) — 飞书上的深度源码拆解（如《Claude Code 整体架构》+ 源码 MAP）。

## 约定

- **不复制源码**：源码留在 `life-os/intake/...`，这里只做索引、"看点"和导读链接（遵循 life-os 仓库"不把源码树复制进来"的约定）。
- 想直接读源码：`资料/agent-frameworks/<agent>/`（软链接，等价于 `life-os/intake/github-trending/agent-frameworks/<agent>/`）。
